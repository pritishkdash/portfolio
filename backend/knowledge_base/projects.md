# Projects

## RAG Chatbot
A production-ready Retrieval-Augmented Generation (RAG) chatbot built with ChromaDB, LangChain, and Groq. Features streaming responses and context-aware answers.

**Tech Stack:** Python, FastAPI, ChromaDB, LangChain, Groq, Docker

## Real-time Object Detection
YOLOv8-based object detection system optimized with TensorRT for edge deployment. Achieves 60fps inference on NVIDIA Jetson devices.

**Tech Stack:** Python, YOLO, TensorRT, OpenCV, Docker, Kubernetes

## ML Platform
End-to-end ML platform with experiment tracking, model registry, feature store, and automated CI/CD pipelines for model deployment.

**Tech Stack:** MLflow, Kubeflow, AWS, Terraform, FastAPI, React

## Sentiment Analyzer
Fine-tuned BERT model for multi-class sentiment analysis with real-time API and batch processing capabilities.

**Tech Stack:** Python, PyTorch, Transformers, FastAPI, React

## Feature Store
Centralized feature store with point-in-time correctness guarantees for ML training and serving.

**Tech Stack:** Feast, Apache Spark, Redis, GCP
